الوصف:
-------
IPLine هي أداة شاملة لفحص عناوين IP والمواقع الإلكترونية وجمع المعلومات
الأمنية والتقنية عنها. تقوم الأداة بفحص شامل يشمل:

- معلومات WHOIS
- سجلات DNS
- فحص المنافذ
- فحص الثغرات الأمنية
- تحليل شهادات SSL
- تحديد الموقع الجغرافي
- البحث في قواعد البيانات المسربة

المتطلبات:
----------
1. Python 3.7 أو أحدث
2. نظام التشغيل: Windows/Linux/macOS
3. اتصال بالإنترنت
4. برنامج Nmap مثبت على النظام

التثبيت:
--------
1. تأكد من تثبيت Python على نظامك
2. قم بتشغيل ملف install_requirements.bat (Windows) أو:
   pip install -r requirements.txt
3. قم بتحميل وتثبيت Nmap من: https://nmap.org/download.html

طريقة الاستخدام:
---------------

الاستخدام الأساسي:
python ipline.py [الهدف]

أمثلة:
------
# فحص موقع إلكتروني
python ipline.py example.com

# فحص عنوان IP
python ipline.py 192.168.1.1

# فحص مع حفظ النتائج
python ipline.py example.com -o report.json

# فحص مع عرض تفاصيل إضافية
python ipline.py example.com -v

الخيارات المتاحة:
----------------
-h, --help          عرض رسالة المساعدة
-o, --output        تحديد ملف الإخراج
-v, --verbose       عرض تفاصيل إضافية أثناء الفحص

مخرجات الأداة:
--------------
تقوم الأداة بإنشاء تقرير شامل يحتوي على:

1. المعلومات الأساسية:
   - اسم الهدف
   - عنوان IP
   - حالة الاتصال

2. معلومات WHOIS:
   - معلومات التسجيل
   - تواريخ الإنشاء والانتهاء
   - خوادم الأسماء
   - معلومات المسجل

3. سجلات DNS:
   - سجلات A, AAAA, MX, NS, TXT, CNAME, SOA
   - خوادم البريد
   - خوادم الأسماء

4. فحص المنافذ:
   - المنافذ المفتوحة
   - الخدمات المكتشفة
   - إصدارات البرامج
   - نظام التشغيل المحتمل

5. الثغرات الأمنية:
   - ثغرات SQL Injection
   - ثغرات XSS
   - ثغرات Directory Traversal
   - Headers الأمنية المفقودة
   - تسريب معلومات الخادم

6. تحليل SSL:
   - معلومات الشهادة
   - تواريخ الصلاحية
   - خوارزمية التوقيع
   - الجهة المصدرة

7. الموقع الجغرافي:
   - البلد والمنطقة
   - المدينة
   - الإحداثيات
   - مزود الخدمة
   - المنطقة الزمنية

8. البحث في البيانات المسربة:
   - التحقق من وجود البيانات في التسريبات
   - عدد التسريبات
   - تاريخ آخر تسريب

ملفات الإخراج:
--------------
- تقرير JSON مفصل: ipline_report_[target]_[timestamp].json
- عرض ملون في الطرفية مع تصنيف الثغرات حسب الخطورة

تصنيف الثغرات:
--------------
- عالية (High): ثغرات خطيرة تتطلب إصلاح فوري
- متوسطة (Medium): ثغرات مهمة يُنصح بإصلاحها
- منخفضة (Low): ثغرات بسيطة أو تحسينات أمنية

ملاحظات مهمة:
--------------
1. استخدم الأداة فقط على المواقع التي تملكها أو لديك إذن لفحصها
2. بعض عمليات الفحص قد تستغرق وقتاً طويلاً حسب حجم الهدف
3. تأكد من وجود اتصال مستقر بالإنترنت
4. قد تحتاج صلاحيات المدير لبعض عمليات فحص المنافذ
5. احترم قوانين الأمن السيبراني في بلدك

استكشاف الأخطاء:
-----------------
1. خطأ "Module not found":
   - تأكد من تثبيت جميع المكتبات المطلوبة
   - قم بتشغيل: pip install -r requirements.txt

2. خطأ "Nmap not found":
   - تأكد من تثبيت Nmap وإضافته لمتغير PATH

3. خطأ "Permission denied":
   - قم بتشغيل الأداة بصلاحيات المدير
   - في Linux: sudo python ipline.py [target]

4. خطأ "Connection timeout":
   - تحقق من اتصال الإنترنت
   - تأكد من أن الهدف متاح ويقبل الاتصالات

5. خطأ "Invalid target":
   - تأكد من صحة عنوان IP أو اسم النطاق
   - تحقق من كتابة الهدف بشكل صحيح
